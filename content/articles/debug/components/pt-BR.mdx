### Exemplo JavaScript

```javascript
function fibonacci(n) {
  if (n <= 1) return n
  return fibonacci(n - 1) + fibonacci(n - 2)
}

const resultado = fibonacci(10)
console.log(`Fibonacci(10) = ${resultado}`)
```

## Exemplo de Código Complexo

Testando syntax highlighting com um exemplo mais complexo:

```typescript
class ServicoDedados<T> {
  private cache = new Map<string, T>()

  constructor(private urlBase: string) {}

  async obter(id: string): Promise<T> {
    if (this.cache.has(id)) {
      return this.cache.get(id)!
    }

    const resposta = await fetch(`${this.urlBase}/${id}`)
    const dados = await resposta.json()

    this.cache.set(id, dados)
    return dados
  }

  limparCache(): void {
    this.cache.clear()
  }
}

const servicoUsuario = new ServicoDeados<Usuario>('/api/usuarios')
const usuario = await servicoUsuario.obter('123')
```
