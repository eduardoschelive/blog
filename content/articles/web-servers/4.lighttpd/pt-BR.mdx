# Lighttpd - Leve e Extremamente Rápido

Lighttpd (pronuncia-se "lighty") é um servidor web projetado para velocidade e eficiência. Com uma pequena pegada de memória e otimizado para ambientes de alta performance, é uma excelente escolha para cenários com recursos limitados.

## O que é Lighttpd?

Lighttpd é um servidor web de código aberto otimizado para ambientes críticos em velocidade, mantendo-se em conformidade com padrões, seguro e flexível. Foi criado por Jan Kneschke em 2003 e foi usado por sites de alto tráfego como YouTube e Wikipedia.

## Recursos Principais

- **Baixa Pegada de Memória**: Usa significativamente menos memória que Apache ou Nginx
- **Arquitetura Orientada a Eventos**: Gerencia muitas conexões simultâneas eficientemente
- **Suporte FastCGI**: Excelente suporte para conteúdo dinâmico
- **Reescrita de URLs**: Motor de reescrita poderoso similar ao mod_rewrite
- **Balanceamento de Carga**: Capacidades de balanceamento de carga integradas
- **Codebase Pequeno**: Fácil de auditar e proteger

## Instalação

Começar com o Lighttpd é simples:

```bash
# Ubuntu/Debian
sudo apt update
sudo apt install lighttpd

# CentOS/RHEL
sudo yum install lighttpd

# Iniciar o serviço
sudo systemctl start lighttpd
sudo systemctl enable lighttpd
```

## Configuração Básica

O arquivo de configuração principal está localizado em `/etc/lighttpd/lighttpd.conf`:

```lighttpd
server.document-root = "/var/www/html"
server.port = 80

mimetype.assign = (
    ".html" => "text/html",
    ".txt" => "text/plain",
    ".jpg" => "image/jpeg",
    ".png" => "image/png"
)
```

## Casos de Uso

O Lighttpd se destaca em cenários específicos:

- Sistemas embarcados com recursos limitados
- Servindo conteúdo estático em alta velocidade
- Servidores de aplicação FastCGI
- Ambientes de desenvolvimento
- Dispositivos IoT

## Otimização de Performance

O Lighttpd já é rápido, mas você pode otimizá-lo ainda mais:

```lighttpd
server.max-connections = 1024
server.max-worker = 4
server.max-fds = 2048

# Habilitar compressão
compress.filetype = ("text/html", "text/css", "text/javascript")
```

## Conclusão

Se você precisa de um servidor web leve e rápido com baixo consumo de recursos, o Lighttpd é uma excelente escolha. É particularmente adequado para sistemas embarcados e servir conteúdo estático de alta performance.
