# NGINX Unit - Servidor de Aplicação Dinâmico Universal

NGINX Unit é um servidor de aplicação leve e de código aberto que pode executar aplicações web escritas em múltiplas linguagens de programação. Oferece configuração dinâmica e atualizações sem downtime através de uma API RESTful.

## O que é NGINX Unit?

NGINX Unit é um servidor web e de aplicação dinâmico projetado para executar aplicações em múltiplas linguagens. Criado pela equipe do NGINX, fornece uma solução unificada para servir aplicações Python, PHP, Go, Perl, Ruby, JavaScript (Node.js) e Java.

## Recursos Principais

- **Suporte Poliglota**: Execute apps em Python, PHP, Go, Ruby, Perl, Node.js, Java
- **Configuração Dinâmica**: Atualize config via API REST sem reinicializações
- **Zero Downtime**: Aplique mudanças sem derrubar conexões
- **Gerenciamento de Processos**: Spawning automático de processos e gestão de ciclo de vida
- **Suporte TLS**: HTTPS integrado com suporte SNI
- **WebSocket**: Suporte completo a WebSocket
- **Leve**: Pegada mínima de recursos

## Instalação

Instalando NGINX Unit:

```bash
# Ubuntu/Debian
sudo apt update
sudo apt install unit

# CentOS/RHEL
sudo yum install unit

# Instalar módulos de linguagem
sudo apt install unit-dev unit-python3.11 unit-php unit-go
```

## Configuração Básica

Configure o Unit via sua API REST:

```bash
# Ver configuração atual
curl http://localhost:8443/config

# Enviar configuração
curl -X PUT --data-binary @config.json \
     --unix-socket /var/run/control.unit.sock \
     http://localhost/config
```

## Executando uma Aplicação Python

Exemplo de configuração para uma app WSGI Python:

```json
{
    "listeners": {
        "*:8080": {
            "pass": "applications/python_app"
        }
    },
    "applications": {
        "python_app": {
            "type": "python 3.11",
            "path": "/var/www/python_app",
            "module": "wsgi",
            "callable": "application"
        }
    }
}
```

## Executando uma Aplicação PHP

Configuração para aplicações PHP:

```json
{
    "listeners": {
        "*:8080": {
            "pass": "applications/php_app"
        }
    },
    "applications": {
        "php_app": {
            "type": "php",
            "root": "/var/www/php_app",
            "index": "index.php"
        }
    }
}
```

## Executando Aplicações Node.js

Configuração para Node.js:

```json
{
    "listeners": {
        "*:8080": {
            "pass": "applications/node_app"
        }
    },
    "applications": {
        "node_app": {
            "type": "external",
            "working_directory": "/var/www/node_app",
            "executable": "/usr/bin/node",
            "arguments": ["app.js"]
        }
    }
}
```

## Gerenciamento de Processos

Configure limites e escala de processos:

```json
{
    "applications": {
        "python_app": {
            "type": "python 3.11",
            "path": "/var/www/python_app",
            "module": "wsgi",
            "processes": {
                "max": 10,
                "spare": 2,
                "idle_timeout": 20
            }
        }
    }
}
```

## Configuração TLS

Habilite HTTPS:

```json
{
    "certificates": {
        "exemplo.com": {
            "key": "/path/to/key.pem",
            "chain": "/path/to/cert.pem"
        }
    },
    "listeners": {
        "*:443": {
            "pass": "applications/app",
            "tls": {
                "certificate": "exemplo.com"
            }
        }
    }
}
```

## Conclusão

NGINX Unit fornece uma abordagem moderna para servir aplicações com sua configuração dinâmica e suporte multi-linguagem. É perfeito para microsserviços e ambientes containerizados.
