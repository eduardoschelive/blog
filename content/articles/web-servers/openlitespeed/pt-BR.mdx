# OpenLiteSpeed - Servidor Web Open Source de Alta Performance

OpenLiteSpeed é um servidor HTTP de código aberto, leve e de alta performance desenvolvido pela LiteSpeed Technologies. É projetado para lidar com milhares de conexões simultâneas com uso mínimo de recursos.

## O que é OpenLiteSpeed?

OpenLiteSpeed é a edição de código aberto do LiteSpeed Web Server Enterprise. Fornece a maioria dos recursos da versão comercial permanecendo completamente gratuito. É particularmente popular para hospedagem WordPress devido às suas excelentes capacidades de cache.

## Recursos Principais

- **Arquitetura Orientada a Eventos**: Gerencia conexões simultâneas massivas eficientemente
- **Cache Integrado**: LSCache para ganhos incríveis de performance
- **Compatível com Apache**: Lê arquivos .htaccess e regras mod_rewrite
- **Suporte HTTP/3**: Suporte aos protocolos QUIC e HTTP/3
- **WebAdmin GUI**: Interface de administração baseada em web
- **Anti-DDoS**: Proteção integrada contra ataques
- **Baixa Memória**: Pegada mínima de memória

## Instalação

Instalando OpenLiteSpeed:

```bash
# Ubuntu/Debian
wget -O - https://repo.litespeed.sh | sudo bash
sudo apt update
sudo apt install openlitespeed

# Iniciar o serviço
sudo systemctl start lsws
sudo systemctl enable lsws

# Definir senha do admin
sudo /usr/local/lsws/admin/misc/admpass.sh
```

Acesse o WebAdmin em: `https://seu-servidor:7080`

## Configuração Básica

OpenLiteSpeed pode ser configurado via:

1. **WebAdmin GUI** (porta 7080)
2. **Arquivos de configuração** em `/usr/local/lsws/conf/`

### Configuração de Virtual Host

Através do WebAdmin:
- Navegue para Virtual Hosts → Add
- Defina document root, domínio e outras configurações
- Aplique as mudanças e reinicie gracefully

## Otimização para WordPress

OpenLiteSpeed é excelente para WordPress:

```bash
# Instalar plugin LSCache
# No admin do WordPress, instale o plugin "LiteSpeed Cache"

# Instalar PHP LSAPI
sudo apt install lsphp80 lsphp80-common lsphp80-mysql
```

Configure o virtual host para usar LSAPI:

```
External App → Add → Type: LiteSpeed SAPI App
Name: lsphp80
Address: uds://tmp/lshttpd/lsphp.sock
```

## Ajuste de Performance

Otimize OpenLiteSpeed para alto tráfego:

```
Server Configuration → Tuning:
- Max Connections: 10000
- Max Keep-Alive Requests: 1000
- Connection Timeout: 60
- Max Cached Small File Size: 4096

Habilitar HTTP/3:
Listeners → SSL → QUIC: Enabled
```

## Configuração LSCache

Habilite e configure LSCache para máxima performance:

```
Virtual Hosts → [Seu VHost] → Rewrite:
Enable Rewrite: Yes
Rewrite Rules: WordPress padrão

Cache Configuration:
Enable Cache: Yes
Cache Storage Path: /usr/local/lsws/cachedata
```

## Configuração SSL/TLS

Configure HTTPS com Let's Encrypt:

```bash
# Instalar certbot
sudo apt install certbot

# Obter certificado
sudo certbot certonly --webroot \
  -w /usr/local/lsws/Example/html \
  -d exemplo.com -d www.exemplo.com

# Configurar no WebAdmin:
# Listeners → SSL → SSL Private Key & Certificate
```

## Regras de Reescrita

OpenLiteSpeed suporta regras de reescrita estilo Apache:

```apache
RewriteEngine On
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^(.*)$ index.php/$1 [L]
```
