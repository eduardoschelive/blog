# Traefik - Roteador de Borda Moderno Cloud-Native

Traefik é um proxy reverso HTTP moderno e balanceador de carga que facilita a implantação de microsserviços. Construído para ambientes dinâmicos, ele descobre serviços automaticamente e se configura dinamicamente.

## O que é Traefik?

Traefik é um roteador de borda de código aberto que se integra perfeitamente com seus componentes de infraestrutura existentes e se configura automaticamente e dinamicamente. Criado pensando em microsserviços e contêineres, é o companheiro perfeito para Docker, Kubernetes e outras plataformas de orquestração.

## Recursos Principais

- **Descoberta Automática de Serviços**: Detecta serviços automaticamente do Docker, Kubernetes, Consul, etc.
- **Integração Let's Encrypt**: Geração e renovação automática de certificados HTTPS
- **Balanceamento de Carga**: Múltiplos algoritmos de balanceamento de carga
- **Suporte WebSocket**: Suporte completo para conexões WebSocket
- **Sistema de Middleware**: Extensível com autenticação, limitação de taxa, manipulação de headers
- **API e Dashboard**: Dashboard em tempo real e API REST

## Instalação com Docker

Começando com Traefik usando Docker:

```bash
# Criar um docker-compose.yml
version: '3'

services:
  traefik:
    image: traefik:v2.10
    command:
      - "--api.insecure=true"
      - "--providers.docker=true"
      - "--entrypoints.web.address=:80"
    ports:
      - "80:80"
      - "8080:8080"
    volumes:
      - /var/run/docker.sock:/var/run/docker.sock
```

## Exemplo de Configuração

Traefik usa labels para configurar roteamento:

```yaml
services:
  myapp:
    image: myapp:latest
    labels:
      - "traefik.enable=true"
      - "traefik.http.routers.myapp.rule=Host(`myapp.exemplo.com`)"
      - "traefik.http.routers.myapp.entrypoints=web"
```

## Integração com Kubernetes

Traefik funciona nativamente com Kubernetes usando IngressRoute:

```yaml
apiVersion: traefik.containo.us/v1alpha1
kind: IngressRoute
metadata:
  name: myapp
spec:
  entryPoints:
    - web
  routes:
    - match: Host(`myapp.exemplo.com`)
      kind: Rule
      services:
        - name: myapp
          port: 80
```

## Configuração de Middleware

O sistema de middleware do Traefik adiciona recursos poderosos:

```yaml
labels:
  - "traefik.http.middlewares.auth.basicauth.users=user:$$apr1$$..."
  - "traefik.http.middlewares.ratelimit.ratelimit.average=100"
  - "traefik.http.routers.myapp.middlewares=auth,ratelimit"
```

## Conclusão

Traefik é a escolha ideal para aplicações modernas cloud-native. Sua descoberta automática de serviços e configuração dinâmica o tornam perfeito para arquiteturas de contêineres e microsserviços.
