# Categoria Debug

Esta categoria é dedicada a testar e debugar todos os componentes e funcionalidades do sistema do blog. Aqui você encontrará artigos que demonstram diferentes componentes MDX, estilos e funcionalidades.

## Propósito

A categoria debug serve como um ambiente de testes para:

- Renderização de componentes MDX
- Syntax highlighting com Shiki
- Tipografia e formatação de texto
- Otimização de performance
- Recursos de acessibilidade
- Design responsivo

Use os artigos desta categoria para verificar se todos os componentes estão funcionando corretamente e para testar novos recursos antes de colocar em produção.
